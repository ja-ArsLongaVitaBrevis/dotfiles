## CREATING VIRTUAL ENV:

CREATING

```sh
python3 -m venv /path/to/new/virtual/environment
```

ACTIVATION

```sh
source <venv>/bin/activate
```

DEACTIVATION

```sh
deactivate
```

INSTALLING DEPENDENCIES

```sh
python3 -m pip install -r /path/to/new/virtual/environment
```
